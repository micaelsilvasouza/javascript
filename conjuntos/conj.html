Criando um conjuto set

Um conjunto JavaScript é uma coleção de valores exclusivos. Cada valor só pode ocorrer uma vez em um Set.

Métodos de conjunto essenciais

Método Descrição
new Set() Cria um novo conjunto
add() Adiciona um novo elemento ao conjunto
delete() Remove um elemento de um conjunto
has() Retorna true se existir um valor no conjunto
forEach() Invoca um retorno de chamada para cada elemento no conjunto
values() Retorna um iterador com todos os valores em um conjunto
Propriedade Descrição
size Retorna o número de elementos em um conjunto
